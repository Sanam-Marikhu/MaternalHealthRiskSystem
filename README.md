# Maternal Health Risk Triage System

A machine learning-based web application that predicts maternal health risk levels (Low, Medium, High) using patient health parameters.

## Technologies
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

## Features
- Risk prediction
- Interactive web interface
- Dataset visualization
- Model persistence using Joblib

## Run

pip install -r requirements.txt

python train_model.py

streamlit run app.py
